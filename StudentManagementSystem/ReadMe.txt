Student Management System - Java Console Application
===================================================

Description:
------------
This is a simple console-based Student Management System written in Java. 
It allows you to add, update, delete, search, and view student records. 
Student data is saved to a text file (`studentmanagement.txt`) in the project folder.

How to Run:
-----------
1. Make sure you have Java installed on your system.
   - To check, open Command Prompt and type: `java -version`

2. Open Command Prompt and navigate to the project directory:
   ```
   cd c:\Users\joymo\Desktop\javaaa\CEC_JAVAMAIN
   ```

3. Compile the Java source file:
   ```
   javac studentmanagement.java


4. Run the program:
   ```
   java studentmanagement
   ```

5. Follow the on-screen menu to manage student records.

Notes:
------
- All student data is stored in `studentmanagement.txt` in the same folder.
- If the file does not exist, it will be created automatically.
- To reset all data, simply delete `studentmanagement.txt`.
